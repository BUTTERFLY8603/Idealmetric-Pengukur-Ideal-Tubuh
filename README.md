# Idealmetric-Pengukur-Ideal-Tubuh
Body Mass Index (BMI) adalah indikator penting untuk menilai apakah berat badan Anda ideal, kurang, atau berlebih. Dengan mengetahui BMI, Anda dapat mengambil langkah yang tepat untuk menjaga kesehatan dan mencapai berat badan ideal. Kalkulator ini memberikan hasil yang akurat dan mudah dipahami untuk membantu Anda memahami status kesehatan tubuh.
